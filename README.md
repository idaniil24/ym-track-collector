# YM Track Collector (Console)

Скрипт для сбора треков из плейлистов Яндекс Музыки через DevTools Console.  
Работает с виртуализированным списком: **нужно пролистать плейлист** (можно быстро), затем нажать **Finalize capture**, и скачать `.txt/.csv`.

Автор/тег: idaniil24

---

## Как использовать
1. Открой плейлист в Яндекс Музыке (web).
2. Открой DevTools:
   - Chrome/Edge: `F12` → вкладка **Console**
3. Открой файл [`ym-track-collector.js`](./ym-track-collector.js) в этом репозитории.
4. Скопируй весь код (Ctrl+A → Ctrl+C) и вставь в Console → Enter.
5. Скроль плейлист вниз.
6. Когда закончил — перестань скроллить и нажми **Finalize capture** в появившейся панели.
7. Скачай `tracks.txt` или `tracks.csv`.

---

## Форматы
- TXT: `Artist - Title`
- CSV: `Artist,Title,Duration`

---

## Примечания
- Яндекс.Музыка рендерит треки порциями (виртуализация), поэтому если не прокрутить — треки не попадут в DOM и не будут собраны.
- Если прокрутил слишком быстро — **Finalize capture** добирает хвост после остановки.
